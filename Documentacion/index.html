1. Asignación de roles
ID: REQ004

Se debe tener un apartado donde se le asignen los roles a los usuarios:

  1. Aprendiz
  2. Usuario SENA
  3. Instructor
  4. Coordinador

 

2. Datos obligatorios
ID: REQ002

Los datos que son fundamentales para el registro, identificación reconocimiento y comunicación son:

  1. documento.
  2. nombre
  3. apellido
  4. email

 

3. Emails únicos
ID: REQ003
Los correo de las personas registradas deben ser únicos por cada usuario.
4. Registro de usuarios
ID: REQ001

El sistema muestra un formulario que permita el registro de los usuarios:

Se recopila los datos:

  1. documento.
  2. nombre
  3. apellido
  4. email
